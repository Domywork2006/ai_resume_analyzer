# AI Resume Analyzer

AI Resume Analyzer is a modern Flutter application that helps students, fresh graduates, and job seekers manage and improve their resumes using AI-powered analysis and Firebase services.

The application provides secure Google Authentication, a personalized dashboard, resume management capabilities, and a foundation for future ATS scoring and AI-driven resume feedback.

## Installation

### Clone Repository

```bash
git clone https://github.com/Domywork2006/ai_resume_analyzer.git
```

### Navigate to Project

```bash
cd ai_resume_analyzer
```

### Install Dependencies

```bash
flutter pub get
```

### Run Application

```bash
flutter run
```

---

## Firebase Configuration

1. Create a Firebase project.
2. Register the Android application.
3. Enable Google Authentication.
4. Add SHA-1 and SHA-256 fingerprints.
5. Download and replace `google-services.json`.
6. Configure FlutterFire.
7. Run the application.
